# 🏨 Shazaib hostel - Hostel Management System

A complete Django-based hostel management system for managing rooms, bookings, and customer inquiries.

## ✨ Features

- 🛏️ **Room Management** - Add, edit, and manage hostel rooms
- 📸 **Room Gallery** - Multiple images per room with slider
- 📅 **Online Booking** - Real-time booking system with price calculator
- 💬 **Contact Form** - Customer inquiry management
- 👑 **Admin Panel** - Complete control over all data
- 📱 **Responsive Design** - Works on mobile, tablet, and desktop
- 🎨 **Modern UI** - Beautiful gold-themed design

## 🛠️ Technology Stack

- **Backend:** Django 4.2+
- **Frontend:** HTML5, CSS3, JavaScript
- **Database:** SQLite3 / PostgreSQL
- **Styling:** Custom CSS with gold theme
- **Icons:** Font Awesome 6
- **Fonts:** Google Fonts (Playfair Display, DM Sans)
